# Fake News Detection System

A Machine Learning based Fake News Detection web application built using Flask.

## Features

- Detects Fake and True News
- Uses four Machine Learning models:
  - Logistic Regression
  - Decision Tree
  - Random Forest
  - Gradient Boosting
- Web interface built with Flask

## Technologies Used

- Python
- Flask
- Pandas
- Scikit-learn
- HTML
- CSS

## Project Structure

FakeNewsDetection

## Run the Project

```bash
pip install -r requirements.txt
python app.py
```

Open:

```
http://127.0.0.1:5000
```
